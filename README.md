# Netflix Clone (HTML & CSS)

This project is a Netflix-inspired user interface built using HTML and CSS as part of my frontend development journey.

The goal was to understand how real-world web applications are structured and how UI components come together to create a clean and interactive experience.

## Overview

The project recreates a simplified streaming platform layout, including a hero section, navigation bar, and categorized content rows. It focuses on layout, spacing, and basic interactivity rather than functionality.

## Features

* Semantic HTML structure
* Navigation bar with menu items
* Hero section with title and action buttons
* Content sections (Popular & Trending)
* Hover effects for interactive feel
* Consistent dark theme

## Tech Stack

* HTML
* CSS

## Key Learnings

* Structuring pages using semantic HTML
* Using Flexbox for layout and alignment
* Building reusable UI components
* Improving visual hierarchy through spacing and sizing
* Adding interactivity using hover and transitions

## Project Status

This is a learning project focused on frontend fundamentals.
Next steps include adding JavaScript for dynamic behavior and eventually integrating AI-driven features.
